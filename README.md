# Can Do Urco (SDL Library)

**Can Do Urco** (Galego: *Hell's Dog*) — pronounced /kan do ˈuɾko/

| Word | IPA | Approximate English guide |
| ---- | --- | ------------------------- |
| Can | /kan/ | like "con" in *con artist*, but with an open *a* |
| Do | /do/ | like "doh" in *do-re-mi* |
| Urco | /ˈuɾko/ | "OOR-ko" — *u* as in *flute*, *r* is a single tap (not English *r*), *co* as in *co-op* |

This is a direct import library for SDL for use in the Can Do Urco engine.
